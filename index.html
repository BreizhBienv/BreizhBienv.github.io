Custom Mathematics Library

This is a custom mathematics library made during my studies at Isart Digital Montréal (2021-2023).

It contains vectors from 2 to 4 dimensions, matrices from 3 to 4 dimensions, quaternions and a class vertex that is often used for 3D rendering.

It also contains a test catch using DirectX's library to validate the reliability of the functions.
